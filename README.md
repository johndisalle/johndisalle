## MSP Compliance Tracker

A lightweight, browser-based tool for IT Managed Services Providers to track contracted client obligations.

### Features

- **Client Management** - Add clients with contract details and contact info
- **Obligation Tracking** - Define recurring obligations (weekly, bi-weekly, monthly, quarterly, annually)
- **Dashboard** - At-a-glance view of overdue, upcoming, and recently completed items
- **Calendar View** - Monthly calendar showing all obligations with color-coded status
- **Easy Check-Off** - Mark obligations complete with who did it, when, and optional notes
- **Team Management** - Maintain a list of technicians for quick assignment
- **Data Portability** - Export/import data as JSON for backup

### Quick Start

1. Open `index.html` in any modern browser
2. Click **"Load Sample Data"** in Settings to try it out with example clients
3. Or add your own clients and obligations right away

All data is stored in your browser's localStorage - no server required.
